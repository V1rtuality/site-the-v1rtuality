Привет всем, я хочу представить свою IDE - ну или же среда разработки. 🚀 LuaScriptEdition IDE Эта первая версия IDE, сам он не использует почти IDE, это как блокнот на максималках Python ✨ Особенности В этой среде бывает подсвечиваеться синтаксис, но только слова в кавычках. Это тоесть - не будет ни какой подстветки как Visual Studio Code с плагином Pylance и другими которые подсвечивают синтаксис и их ошибки. По данным тестирования, файл ест немного оперативной памяти. В основном это 20-30 мб. Интерфейс был сделан почти как у Visual Studio code, - ПОЧТИ. Терминал. НО НЕ СОВСЕМ ТЕРМИНАЛ, в большинстве случаях это просто CMD для тестов консольных приложении. Терминал можно двигать вверх и вниз, что бы было удобно работать и не мешал эта консоль или же терминал на минималках. Создание файлов, Удаление файлов и папок, Создание папок, все это можно сделать, с помощью правой кнопкой мыши по слову PROJECT. Run как и во всех IDE. Сохранение файла прямо в IDE, впринцепи что и тоже базово. ⚠️ ВАЖНО ПОДМЕТИТЬ Эта IDE может использоваться и для других языков, даже для С++, все иза моей не внимательности... Но в будущих обновлениях я поменяю интерфейс, и файлы будут создаваться с помощью клика, а не ввода в строку с помектой .lua, потому что опять же - могут создавать файлы с пометкой .py, .html, .js и другие. Файлы OpenSource.py используеться для исходного кода самой IDE. ВАЖНО! Файл с закрытым кодом (.exe) ЭТО НЕ ВИРУС! Потому что эта IDE была написана на Python, а как мы знаем, файлы с пометкой .py не запустяться у обычных юзеров, а у тех у кого есть установленный яп Python. Поэтому используеться формат файла .exe. 📥 Установка ** В разделе Releases найдите файл "luascriptedition.exe" скачиваете его и пользуйтесь, сама программа не требует установки.
